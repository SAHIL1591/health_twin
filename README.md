# Digital Twin Health

Digital Twin Health is an Android-based health simulation application that models lifestyle factors to estimate life expectancy and visualize potential health improvements.

## 🎯 Objective
To demonstrate the concept of a personal digital twin in healthcare by simulating the impact of:
- BMI
- Smoking habits
- Alcohol consumption
- Exercise frequency
- Sleep duration

## 🛠 Tech Stack
- Java
- Android SDK
- Navigation Component
- Material Design 3
- Lottie Animation

## 📌 Current Features
- Splash screen with animated DNA visualization
- Consent and disclaimer validation
- Structured navigation architecture

## 🚀 Planned Features
- Profile setup with automatic BMI calculation
- Lifestyle simulation engine
- Risk factor breakdown visualization
- Goal tracking module
