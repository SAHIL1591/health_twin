# Digital Twin Health

Digital Twin Health is an Android application that estimates life expectancy based on lifestyle factors such as BMI, smoking, alcohol consumption, exercise, and sleep.

## Features
- Splash screen with Lottie DNA animation
- Consent and disclaimer validation
- Profile setup with BMI calculation
- Lifestyle simulation (coming soon)
- Risk breakdown visualization (coming soon)

## Tech Stack
- Java
- Android SDK
- Navigation Component
- Material Design 3
- Lottie Animation
